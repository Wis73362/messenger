# Nova Messenger (Node.js + JavaScript)

Полноценный realtime-мессенджер с уникальным UI и главной страницей, вдохновлённый идеями Discord и Telegram.

## Что уже реализовано

- Регистрация и вход.
- Главная (landing) страница с описанием и входом в приложение.
- Каналы (как серверные комнаты).
- Личные чаты (DM).
- Отправка сообщений в реальном времени (Socket.IO).
- Ответы на сообщения (reply).
- Реакции на сообщения (toggle).
- Редактирование и удаление собственных сообщений.
- Закрепление сообщений.
- Индикатор набора текста.
- Онлайн-статусы пользователей.
- Поиск сообщений по всем доступным чатам.

## Стек

- Node.js
- Express
- Socket.IO
- Vanilla JavaScript (frontend)

## Как запустить

1. Установите зависимости:

```bash
npm install
```

2. Запустите приложение:

```bash
npm start
```

3. Откройте в браузере:

```text
http://localhost:3000
```

## Как пользоваться

1. На главной странице нажмите **«Начать общение»**.
2. Зарегистрируйте первого пользователя.
3. В другой вкладке/браузере зарегистрируйте второго пользователя.
4. Создавайте каналы через кнопку **+ Канал**.
5. Создавайте личные чаты через **+ Личный чат** (по логину).
6. В сообщениях доступны: ответ, реакция, закрепление, редактирование, удаление.

## Важно

Проект сделан как демонстрационно-боевой MVP: архитектура готова к расширению (файлы, звонки, роли, модерация, бот-платформа, шифрование, federation и т.д.).
