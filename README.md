# Nova Messenger (Node.js + JavaScript)

Полноценный realtime-мессенджер с уникальным UI и главной страницей, вдохновлённый идеями Discord и Telegram.

## Что уже реализовано

- Регистрация и вход.
- Главная (landing) страница с описанием и входом в приложение.
- Каналы (как серверные комнаты).
- Личные чаты (DM).
- Отправка сообщений в реальном времени (Socket.IO).
- Ответы на сообщения (reply).
- Реакции на сообщения (toggle).
- Редактирование и удаление собственных сообщений.
- Закрепление сообщений.
- Индикатор набора текста.
- Онлайн-статусы пользователей.
- Поиск сообщений по всем доступным чатам.

## Стек

- Node.js
- Express
- Socket.IO
- Vanilla JavaScript (frontend)

## Как скачать с GitHub

### Вариант 1: Прямая ссылка на ZIP (если кнопки нет)

Если в интерфейсе GitHub не видно **Download ZIP** (например, другой UI/мобильная версия), скачайте архив напрямую:

```text
https://github.com/<OWNER>/<REPO>/archive/refs/heads/<BRANCH>.zip
```

Пример для ветки `main`:

```text
https://github.com/<OWNER>/<REPO>/archive/refs/heads/main.zip
```

После скачивания распакуйте архив и перейдите в папку проекта.

### Вариант 2: Через Git (очень подробно, рекомендуемый способ)

#### 0) Установите Git и проверьте, что он доступен

```bash
git --version
```

Если команда не найдена:
- **Windows**: установите Git for Windows (добавьте Git в PATH во время установки).
- **macOS**: установите Xcode Command Line Tools (`xcode-select --install`) или Git через Homebrew.
- **Linux**: установите Git через менеджер пакетов (`apt`, `dnf`, `pacman` и т.д.).

#### 1) Откройте нужную страницу репозитория на GitHub

На странице репозитория нажмите **Code** и скопируйте ссылку на клонирование:
- HTTPS-формат: `https://github.com/<OWNER>/<REPO>.git`
- SSH-формат: `git@github.com:<OWNER>/<REPO>.git`

> Для большинства пользователей проще начать с HTTPS.

#### 2) Перейдите в папку, куда хотите скачать проект

Например:

```bash
cd ~/projects
```

Или создайте папку заранее:

```bash
mkdir -p ~/projects
cd ~/projects
```

#### 3) Склонируйте репозиторий

**HTTPS:**

```bash
git clone https://github.com/<OWNER>/<REPO>.git
```

**SSH:**

```bash
git clone git@github.com:<OWNER>/<REPO>.git
```

Если нужно скачать в папку с другим именем:

```bash
git clone https://github.com/<OWNER>/<REPO>.git messenger
```

#### 4) Перейдите в папку проекта

```bash
cd messenger
```

Если название папки другое — используйте его вместо `messenger`.

#### 5) Проверьте, что remote настроен правильно

```bash
git remote -v
```

Вы должны увидеть URL вашего GitHub-репозитория для `origin`.

#### 6) (Опционально) Переключитесь на нужную ветку

Показать ветки:

```bash
git branch -a
```

Переключиться на ветку:

```bash
git checkout <BRANCH>
```

#### 7) Установите зависимости Node.js

```bash
npm install
```

#### 8) Запустите проект

```bash
npm start
```

#### 9) Откройте приложение в браузере

```text
http://localhost:3000
```

#### 10) Как обновлять проект позже

Когда репозиторий уже скачан, для получения свежих изменений:

```bash
cd messenger
git pull
npm install
npm start
```

#### Частые проблемы и решения

1. **`Repository not found` / `403` / `Authentication failed`**  
   - Проверьте URL репозитория.  
   - Если репозиторий приватный — авторизуйтесь в GitHub (PAT для HTTPS или SSH-ключ).

2. **`Permission denied (publickey)` при SSH**  
   - SSH-ключ не добавлен в GitHub аккаунт или ssh-agent не запущен.

3. **`git: command not found`**  
   - Git не установлен или не добавлен в PATH.

4. **`npm install` не работает**  
   - Проверьте интернет/прокси и registry (`npm config get registry`).

### Вариант 3: Через GitHub CLI

```bash
gh repo clone <OWNER>/<REPO> messenger
cd messenger
npm install
npm start
```

Если репозиторий приватный, нужна авторизация в GitHub (SSH-ключ, Personal Access Token или `gh auth login`).

## Как запустить

1. Установите зависимости:

```bash
npm install
```

2. Запустите приложение:

```bash
npm start
```

3. Откройте в браузере:

```text
http://localhost:3000
```

## Как пользоваться

1. На главной странице нажмите **«Начать общение»**.
2. Зарегистрируйте первого пользователя.
3. В другой вкладке/браузере зарегистрируйте второго пользователя.
4. Создавайте каналы через кнопку **+ Канал**.
5. Создавайте личные чаты через **+ Личный чат** (по логину).
6. В сообщениях доступны: ответ, реакция, закрепление, редактирование, удаление.
